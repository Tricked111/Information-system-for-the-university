Implementace

Backend

implementaci serveru jsme provedli v jazyce Python a jako framework jsme zvolili Django pro vytvořeni REST API.
Client posílá dotazy na naš server, všechny cesty se nachazejí ve souboru main/urls.py, který následně volá funkce ze souboru views.py.
Funkce souboru views.py se volá funkcionalitu Django, který odesílá dotazy do databáze, manipuluje s daty a odesílá odpověď klientovi.

Frontend

V tomto projektu byl frontend vytvořen v ReactJS s použitím preprocesoru Sass. Ve složce client/src je hlavní soubor App.js a soubor App.scss, který obsahuje hlavní styly pro hlavní stránku. Dále se zde nacházejí složky Components, Pages, Context, Services a Styles.
Složka Services obsahuje pouze CreateRequest.js kde se nachazí funkce pro vytvořeni dotázu na server
Složka Pages obsahují hlavní stránky
Ve složce Components se nachazejí různé moduly pro upravu uživatelů, kurzů a terminů. Taky tady jsou komponenty jako Footer, Navbar(Header), LoadingIcon.
